# fastmac
A single file install script to setup your new macos environment

To install your new system with fastmac run the following command in a new terminal instance:

```bash
bash ./install.sh
``` 

Packages and applications: 
Homebrew
Node.js
Python
Git
GitHub CLI
Nvim w/ LazyVim
Fastfetch
Ghostty
Spotify
Cursor
Postman
Chromium
Aerospace
NerdFonts
JankyBorders

The current .aerospace.toml file initialized Aerospace with a Hyprland style key bind configuration, using the super key manage and move windows and workspaces.
